# Virtualización de Sistemas Operativos: Implementación y Análisis de Máquinas Virtuales mediante VirtualBox y Ubuntu

## Descripción

Este repositorio contiene el Trabajo Práctico Integrador de la asignatura **Arquitectura de Computadoras y Sistemas Operativos** de la **Tecnicatura Universitaria en Programación a Distancia (UTN)**.

El trabajo aborda el estudio de la virtualización de sistemas operativos mediante la utilización de **Oracle VM VirtualBox** y la instalación de **Ubuntu Linux** como sistema operativo invitado sobre **Windows 11**.

## Objetivos

- Comprender los fundamentos de la virtualización.
- Analizar el funcionamiento de las máquinas virtuales.
- Estudiar los distintos tipos de hipervisores.
- Implementar una máquina virtual utilizando VirtualBox.
- Instalar y configurar Ubuntu Linux en un entorno virtualizado.
- Evaluar ventajas, limitaciones y aplicaciones de la virtualización.

## Tecnologías Utilizadas

- Windows 11 (Sistema Operativo Host)
- Oracle VM VirtualBox
- Ubuntu Linux
- Git y GitHub
- Microsoft Word

## Contenido del Trabajo

### Marco Teórico

- Concepto de Virtualización
- Máquinas Virtuales
- Hipervisores Tipo 1 y Tipo 2
- VirtualBox
- Sistemas Operativos Host y Guest
- Aplicaciones de la Virtualización
- Relación con la Programación y Administración de Sistemas

### Caso Práctico

Implementación de una máquina virtual Ubuntu sobre Windows 11 utilizando VirtualBox:

1. Instalación de VirtualBox.
2. Creación de la máquina virtual.
3. Asignación de recursos:
   - 4 GB RAM
   - 2 núcleos de CPU
   - 25 GB de almacenamiento virtual
4. Instalación de Ubuntu Linux.
5. Pruebas de funcionamiento y conectividad.

## Resultados Obtenidos

Se logró:

- Instalar correctamente Ubuntu Linux en una máquina virtual.
- Ejecutar comandos básicos de Linux.
- Verificar la conectividad a Internet.
- Comprobar el aislamiento entre sistema anfitrión e invitado.
- Analizar el comportamiento de los recursos virtualizados.
